# ExtensionExample

## 설명
    브라우저를 열었을 때 작동하는 확장 프로그램
  
## 기능
    1. 현재 시각 출력
    2. 사용자의 이름 기억
    3. To-Do List를 입력받고 기억한다.
    4. 사용자의 위치에 따라 날씨 정보 출력
